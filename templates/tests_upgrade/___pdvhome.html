Priority Data Sets Viewer

Explores the data sets identified by EU Member States and EFTA countries as priority data sets for environmental reporting. You can filter by environmental domain, environmental legislation and country.

The list of priority data sets has been defined and is being further developed in the framework of INSPIRE Maintenance and Implementation Work Plan 2016-2020 (MIWP).

The Priority Data Sets Viewer provides statistical overviews of the availability of priority data set as well as simplified access to view or download selected data sets.
